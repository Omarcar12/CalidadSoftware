Pruebas y evaluación

Realizar pruebas sistemáticas asegura que el aplicativo cumpla sus objetivos pedagógicos y que los usuarios (estudiantes y docentes) tengan una experiencia correcta.

Unitarias

Pruebas pequeñas que verifican funciones individuales. Son rápidas y fáciles de automatizar.

Integración

Comprueban que módulos o servicios funcionen bien en conjunto.

Usabilidad

Evaluaciones con usuarios reales para medir si la interfaz cumple con objetivos pedagógicos.

Ejemplo de test simple

function testDuplicarNumero() {
    console.assert(duplicarNumero(2) === 4, 'duplicarNumero debería devolver 4');
}

Incluye pruebas automáticas y tests de usabilidad para mejores resultados.
