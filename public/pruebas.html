Pruebas y Evaluación de Software

Tipos de pruebas importantes:

  • Unitarias: Verificar funciones individuales.
  • Integración: Verificar que las funciones trabajen juntas correctamente.
  • Funcionales: Probar que el software cumpla los requisitos.
  • Usabilidad: Evaluar la experiencia del usuario.

Ejemplo de prueba unitaria en JavaScript:

function testDuplicarNumero() {
    console.assert(duplicarNumero(2) === 4, "Error: duplicarNumero no funciona correctamente");
}
