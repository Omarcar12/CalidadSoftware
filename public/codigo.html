Buenas Prácticas de Programación

Ejemplo de código limpio y comentado:

Antes:

function a(x){return x*2}

Después:

// Función que duplica un número
function duplicarNumero(numero){
    return numero * 2;
}

Aplicar buenas prácticas hace que el código sea más legible, mantenible y seguro.
